# Travellingo - AI Powered Travel Planner

Travellingo is a modern, AI-powered travel planning landing page designed to help travelers plan less and explore more. It provides a seamless interface for creating itineraries, discovering local gems, and managing trips.

## Features

- **Smart Trip Planner**: Generate optimized multi-city itineraries in minutes.
- **Live Travel Assistant**: Real-time updates on traffic, delays, and local tips.
- **Local Insider AI**: Discover hidden gems and local favorites that typical guides miss.
- **Group Coordination**: Share plans and stay in sync with your travel companions.

## Tech Stack

- **Frontend**: React.js, HTML5
- **Styling**: Tailwind CSS, PostCSS
- **Build Tool**: Vite
- **Icons**: Lucide, Font Awesome

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production build:
```bash
npm run build
```
